## How to Run the Project Locally

1. Clone the repo: `git clone https://github.com/lvyza/MyPortfolio-laravel-project.git`  
2. Go to the project folder: `cd MyPortfolio-laravel-project`  
3. Install dependencies: `composer install`  
4. Copy `.env.example` to `.env`: `cp .env.example .env` (or manually create `.env` on Windows)  
5. Generate app key: `php artisan key:generate`  
6. Run migrations (if using a database): `php artisan migrate`  
7. Start the server: `php artisan serve`  
8. Open in browser: `http://127.0.0.1:8000`
